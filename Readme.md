## comandos en git

* Check the status of a local repository. **git status**
* Add individual files or globally. **git add** agrega todos los archivos   
* Add comments to the commit. **git commit -m** "Describe tus cambios aquí"
* Upload your changes to the remote repository. **git push origin main** Envía confirmaciones a la rama remota 
* Create, browse, and delete branches. **git branch new-branch** crea una nueva rama **git checkout new-branch** Cambiar a una sucursal
* Roll back a repository to a specific commit. **git log** ver historial de confirmaciones **git checkout commit_id** Regresar temporalmente a una confirmación anterior **git reset --hard commit_id** Restablecer permanentemente una confirmación (PELIGRO: elimina los cambios) **git reset --soft commit_id** Restablecer pero conservar archivos
